LeapMux

Choose how you'd like to connect

💻

Solo

Run LeapMux entirely on this machine. A Hub and Worker start together in a single process — no network setup required. Your data stays local. Ideal for personal use, local development, or trying out LeapMux.

🌐

Distributed

Connect to a remote LeapMux Hub shared across your team. Multiple workers can connect to a centralized hub for collaborative workflows. Requires a Hub server already running and accessible at the URL you provide.

🔒

Full Disk Access Required

Solo mode needs Full Disk Access to traverse directories and files in your home directory. Grant access in System Settings and the app will restart automatically.
